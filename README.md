# AI Explainer EC2

AI Explainer EC2 is a simple web application built using Flask and LangChain. It leverages Groq's high-speed inference and the `llama-3.1-8b-instant` model to explain any topic from the perspective of a specific role, all restricted to a concise, single-sentence output.

## Features
- **Role-based Explanation**: Describe any concept tailored to a specific persona (e.g., Explain 'Black Holes' like a '5-year-old', or 'Quantum Computing' like a 'Software Engineer').
- **Powered by LangChain & Groq**: Uses `init_chat_model` from LangChain configured with Groq Cloud inference API for lightning-fast responses.
- **Simple UI**: A clean, easily understandable web interface using HTML and Flask.

## Prerequisites
- Python 3.8+
- A [Groq API Key](https://console.groq.com)

## Installation & Setup

1. **Navigate to the Project Directory**
   Ensure you are in the correct folder:
   ```bash
   cd 16-ai-explainer-ec2
   ```

2. **Install Dependencies**
   It's generally recommended to activate a virtual environment first. Then, install the required packages:
   ```bash
   pip install flask langchain langchain-groq langchain-core
   ```

3. **Set your API Key**
   The application requires a Groq API key to function. Open `app.py` and assign your Groq API key to the environment variable.
   ```python
   # Inside app.py
   os.environ["GROQ_API_KEY"] = "your_actual_api_key_here"
   ```
   *(Note: For production, it's highly recommended to strictly use `.env` files and `python-dotenv` instead of directly hardcoding API keys.)*

4. **Run the Application**
   Execute the Flask deployment directly:
   ```bash
   python app.py
   ```

5. **Access the Web Interface**
   Open your browser and navigate to:
   [http://127.0.0.1:5000](http://127.0.0.1:5000)

## Usage
1. Enter the persona or `Role` in the first input box (e.g., "Pirate", "Scientist", "Grandma").
2. Enter the `Topic` you want explained in the second input box (e.g., "Machine Learning", "Photosynthesis").
3. Click to submit. The web page will refresh with a custom, single-sentence explanation dynamically generated by the Llama 3 model.
